# Review Artifact Contracts

These schemas define the missing middle layer between detection and final
reporting.

They are first-wave contracts only:

- safe to document
- safe to test for syntax/shape
- not a claim that every runtime path emits them yet

Artifacts:

- `feature_identity.schema.json`
- `bottleneck_candidates.schema.json`
- `fix_options.schema.json`
- `verification_plan.schema.json`
