# Module 04: AI-Assisted Manufacturing Review

This module explains how to talk about AI assistance responsibly in manufacturing-review contexts.

## Learning Goals

- Separate deterministic guidance from verified engineering decisions.
- Understand where AI assistance can help with review speed and summarization.
- Explain why human manufacturing and quality owners stay in the loop.

## Safe Claims

- This repo can help structure review data, summarize context, and highlight likely constraints.
- The new DELMIA-adjacent demos use deterministic heuristics so the reasoning remains auditable.
- Recommendations are guidance for review, not sign-off.

## Demo Steps

1. Show the flow simulation report and call out the rule basis.
2. Show the inspection linkage report and its evidence references.
3. Ask the audience which human approvals would still be required before action.

## Discussion Questions

- Where is AI assistance useful but low risk?
- Where would you insist on engineering validation before acting?
- How would you explain the difference between recommendation and truth to a customer?
