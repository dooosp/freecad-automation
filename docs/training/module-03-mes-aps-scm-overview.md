# Module 03: MES, APS, And SCM Overview

This module gives non-specialists a simple map of MES, APS, and SCM language using the new bracket manufacturing context example.

## Learning Goals

- Distinguish execution, planning, and supply terminology.
- Explain where this repo can support each topic today.
- Explain what is out of scope for this prototype.

## Quick Definitions

- MES / MOM: what is happening on the floor now and what quality/traceability evidence is captured.
- APS: how demand, sequencing, and capacity are balanced before release.
- SCM: how materials, suppliers, and packaging readiness shape production decisions.

## Demo Walkthrough

1. Show `production_line.mes_scope`, `aps_scope`, and `scm_scope`.
2. Show `planning_data.aps_constraints` and `supplier_watchlist`.
3. Show how the flow report translates that context into bottleneck and WIP language.
4. Show how inspection linkage keeps the quality loop attached to the routing.

## Discussion Questions

- Which fields would an MES team care about first?
- Which fields would an APS owner challenge?
- What SCM data would improve the realism of the demo?
