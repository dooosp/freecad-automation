# Module 01: Digital Manufacturing

This module introduces digital manufacturing using `freecad-automation` as a DELMIA-adjacent portfolio demo. It is not official DELMIA training.

## Learning Goals

- Understand how CAD, process planning, line planning, and quality evidence connect.
- Explain why manufacturing teams need more than a single CAD file.
- Recognize the difference between a portfolio demo and a production platform.

## Core Message

Digital manufacturing is not only about 3D geometry. It is about connecting the product, the process, the line, the inspection loop, and the decisions that follow. This repo demonstrates that connection through review packs, process plans, line plans, quality linkage, and readiness artifacts.

## Demo Steps

1. Start with the bracket CAD/config example.
2. Show the manufacturing context JSON and explain the plant, line, routing, BOM, inspection, and planning sections.
3. Show how the production-flow and inspection-linkage demos turn that context into decision-support outputs.
4. Reinforce that the output is guidance, not manufacturing authority.

## Discussion Questions

- Which additional plant data would you need before trusting the bottleneck story?
- Which outputs would matter most to a manufacturing engineer versus a solution consultant?
- What would have to be added before this moved from demo to pilot?
